LOJA VESTUARIO

Este projeto é uma loja virtual desenvolvida com foco em desempenho, usabilidade e organização de produtos.

Utiliza HTML5, CSS3 e JavaScript no front-end para criar uma interface moderna, responsiva e interativa, com recursos como carrossel de produtos, filtros inteligentes e um carrinho de compras dinâmico, que permite adicionar ou remover itens com atualização automática do valor total.

O sistema também conta com um painel administrativo, onde é possível cadastrar, editar e remover produtos de forma prática e segura.

Este projeto representa uma aplicação completa de conhecimentos em desenvolvimento web, integrando tecnologias de front-end e back-end em um ambiente coeso e funcional.

LOJA ONLINE DE MAKEUP

Glamour Rose é um projeto de loja virtual desenvolvido com o objetivo de simular uma experiência real de e-commerce voltado para o público feminino, com foco em cosméticos, beleza e bem-estar.

A proposta do site é unir estética, funcionalidade e desempenho, proporcionando uma navegação fluida e intuitiva para o usuário.

A interface foi construída utilizando HTML5, CSS3 e JavaScript, garantindo um visual elegante e moderno. Conta com carrossel de produtos, categorias organizadas e um carrinho de compras dinâmico, onde é possível adicionar e remover itens com atualização automática do valor total.

O layout é totalmente responsivo, funcionando perfeitamente em computadores, tablets e celulares, oferecendo uma ótima experiência em qualquer dispositivo.
